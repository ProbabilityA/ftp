# FTP-Server

## Features
- Authorization (Common login, anonymous login, quit)
  - USER
  - PASS
  - QUIT
- File operation (List files, change directory, upload and download file)
  - LIST
  - CWD
  - STOR
  - RETR
- Initiative mode and passive mode
  - PORT
  - PASV
- Others (not fully implements)
  - OPTS
  - TYPE
  - SYST

## Dependency
- Java 11
- SpringBoot 2.5.0
- lombok 1.18.20
